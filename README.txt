# xyPanels

http://xypanels.com

http://carlosruiz.me/projects/xypanels/demo/

xyPanels is a twist on popular one-page website designs in that content containers can be stacked horizontally as well as vertically

The /demo/ folder includes a complete example of how this could be used

The /files/ folder includes the necessary js and css files to be used in your own project
